# Assets Folder

## How to Add Your Profile Photo

1. **Add your photo**: Place your profile photo in this folder
2. **Name it**: Name the file `profile.jpg` (or another format like `.png`, `.webp`)
3. **Update index.html**: If you use a different filename, update line in index.html:

```html
<img src="assets/profile.jpg" alt="Profile photo" class="profile-img">
```

Change `profile.jpg` to your actual filename.

## Recommended Photo Specifications

- **Format**: JPG, PNG, or WebP
- **Size**: 300x300 pixels (square photo works best)
- **Quality**: High resolution (500x500px or larger recommended)
- **Style**: Professional headshot or casual portrait

## Example

If your photo is named `my-photo.png`, update the imgto:

```html
<img src="assets/my-photo.png" alt="Profile photo" class="profile-img">
```

That's it! Your photo will appear in the circular frame on the home page.
